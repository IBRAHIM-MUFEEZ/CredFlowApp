## Technical Implementation

### Color Constants
```kotlin
val primaryPurple = 0xFF667EEA.toInt()      // #667EEA
val violetSecondary = 0xFF764BA2.toInt()    // #764BA2
val lightBg = 0xFFEDE8FF.toInt()            // #EDE8FF
val darkText = 0xFF1A0A40.toInt()           // #1A0A40
val mediumText = 0xFF5A4880.toInt()         // #5A4880
val grayText = 0xFF999999.toInt()           // #999999
val settledGreen = 0xFF4CAF50.toInt()       // #4CAF50
val pendingOrange = 0xFFFF9800.toInt()      // #FF9800
```

### Timestamp Format
```kotlin
val timestamp = LocalDateTime.now().format(
    DateTimeFormatter.ofPattern("dd MMM yyyy, HH:mm:ss")
)
// Output: "27 Apr 2026, 14:35:22"
```

### User Name Integration
```kotlin
val userName = profileState.profile?.displayName ?: "Radafiq User"
generator.generateStatement(customer, userName)
```
