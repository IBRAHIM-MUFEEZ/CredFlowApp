# Statement PDF Enhancements

## Overview
Enhanced the statement PDF generator with professional branding, app color scheme, user information, and generation timestamp.

## Key Enhancements

### 1. App Branding & Logo
- **App Name**: "RADAFIQ" displayed prominently at the top
- **Decorative Line**: Purple accent line under app name
- **Professional Header**: Clear visual hierarchy with app branding

### 2. App Color Scheme Integration
All PDF elements now use the Radafiq app's Material3 color palette:

**Primary Colors**:
- Primary Purple: `#667EEA` - Main accent color
- Violet Secondary: `#764BA2` - Secondary accent
- Light Background: `#EDE8FF` - Summary boxes background

**Text Colors**:
- Primary Text: `#1A0A40` - Dark purple for main content
- Secondary Text: `#5A4880` - Medium purple for labels
- Muted Text: `#999999` - Gray for footer

**Status Colors**:
- Settled: `#4CAF50` - Green
- Pending: `#FF9800` - Orange

### 3. User Information
- **Generated By**: User's display name shown in footer
- **Format**: "Generated by: [User Name]"
- **Fallback**: "Radafiq User" if name not available

### 4. Generation Timestamp
- **Timestamp Format**: "dd MMM yyyy, HH:mm:ss"
- **Example**: "27 Apr 2026, 14:35:22"
- **Location**: Bottom of each page
- **Color**: Purple accent color for visibility

### 5. Enhanced Footer
Footer now includes three sections:
```
Left:    Generated by: John Doe
Center:  Generated: 27 Apr 2026, 14:35:22
Right:   Page 1 • Radafiq
```

## Implementation Details

### StatementGenerator Changes
```kotlin

    customer: CustomerSummary,
    generatedByName: String = "Radafiq User"
): Result<Uri>
```

### Footer Method Signature
```kotlin
private fun drawFooter(
    canvas: Canvas,
    pageNumber: Int,
    pageHeight: Int,
    generatedByName: String
)
```

### ShareStatementDialog Integration
```kotlin
val profileRepository = remember { UserProfileRepository() }
val profileState by profileRepository.state.collectAsState()
val userName = profileState.profile?.displayName ?: "Radafiq User"

// Pass to generator
generator.generateStatement(customer, userName)
```

## Technical Notes

- All colors use hex codes from `CredFlowTheme.kt`
- Timestamp uses `LocalDateTime` for precision
- User name fetched from `UserProfileRepository`
- Fallback to "Radafiq User" if profile unavailable
- No external dependencies required
- Works on all Android versions (API 19+)
