# Fonts

Place the following font files here for the PDF statement generator:

- `ArgentumSans-Regular.ttf`  — Argentum Sans Regular
- `ArgentumSans-SemiBold.ttf` — Argentum Sans SemiBold

Download from: https://github.com/evansendra/argentum-sans/releases
(OFL 1.1 licensed — free to bundle)

If these files are absent, the generator falls back to the system sans-serif font automatically.
