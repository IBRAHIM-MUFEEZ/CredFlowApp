## 🚀 How to Use

### For Users
1. Open customer detail screen
2. Tap Share icon (top right)
3. Dialog shows statement preview
4. Tap "Generate & Share"
5. PDF generated and share sheet opens
6. Choose destination (Email, Drive, WhatsApp, etc.)

### For Developers
```kotlin
// Generate statement with user name
val generator = StatementGenerator(context)
val statementUri = generator.generateStatement(
    customer = customer,
    generatedByName = "John Doe"
).getOrThrow()

// Share the PDF
val shareIntent = Intent().apply {
    action = Intent.ACTION_SEND
    putExtra(Intent.EXTRA_STREAM, statementUri)
    type = "application/pdf"
    addFlags(Intent.FLAG_GRANT_READ_URI_PERMISSION)
}
context.startActivity(Intent.createChooser(shareIntent, "Share Statement"))
```

```

    ↓

    ↓

    ↓

    ↓
```
